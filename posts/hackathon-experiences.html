  • HackIllinois

    April 2014

    My very first hackathon experience can be summarized with the following list:


    • Hours of thinking up ideas.
    • Hours trying to set up developer tools.
    • Too much sugar and unhealthiness.
    • Leaving early and not completing a project.
    • Fun, and a new desire to attend more hackathons.

  • MHacks IV

    September 2014

    • 6-7 hour bus ride.
    • First 36 hour hackathon that I made through the whole time.
    • Had an amazing experience that really solidified my interest in hackathons.
    • My interest in web development began here.
    • Made a web UI for a parking meter app.
    • Met great people and friends.

  • IEEE@UIUC Hackathon

    September 2014

    • First 24 hour hackathon.
    • Hosted at the new ECE building at UIUC.
    • Built my first android app.

  • BoilerMake

    October 2014

    • Managed to build two web apps (and demo-ed one) in 36 hours.
    • Built a job search web application and a server that sent coding problems as text messages to your mobile phone. It took text replies as answers to the problems.
    • Had really good food for a hackathon.

  • WildHacks

    November 2014

    • Met some really awesome people.
    • Built another web app. This one helped users with their grammar!
    • Late night coffee runs on Northwestern University's campus

  • PennApps

    January 2015

    • Flew from Chicago to Philidelphia, PA to attend a hackathon at no cost to me. Awesome.
    • Saw some really cool projects, met some cool people.
    • Saw my cousins!

  • HackIllinois

    February 2015

    • The place that started it all, back again! Worked on a web design generating website with some friends.

  • VandyHacks

    March 2015

    • Took a bus for about 6 hours to Vanderbilt in Nashville, TN with cool peeps.
    • Visited Nashville for the first time!
    • Hacked with about 100 others in a startup space that really nice.
    • Built an app that generated music based on a person's mood which we computed through brainwaves using an EEG wearable device.
