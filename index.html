    Projects

You can view the source of most of these projects on my Github. (Note: This section is under construction)

  • Code Texters

    Code Texters allows you to subscribe to a service that sends coding-related questions right to your mobile phone. It only requires your phone number to get started and you can unsubscribe at anytime. Register online to log your stats and see how you stack up against other users. Visit the homepage or see the source.

  • Flick

    Flick is an android application that allows users to physically flick their phone to send webpages from the mobile browser directly to a browser that has the accompanying flick chrome extension installed. Flick was built at a mini-hackathon hosted by IEEE@UIUC. See the source.

  • Grammatica

    Grammatica is a grammar and spelling correction web service that was built at Wildhacks 2014. Users can visit the site and enter any phrase to be analyzed and corrected. See the source.
