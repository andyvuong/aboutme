Projects

The following are featured projects. Please see this link for more.

Safe Route Recommender [Github] [Blog]

A recommender system built on top of adaptive information filtering and machine learning principles to algorithmitically generate the "safest" walking path between two points on UIUC's campus. Started as a CS410 project, continued afterward.

Jumpy [Github]

Jumpy is a Java library being developed for java-based numerical linear algebra. Jumpy will support n-dimensional arrays for numerical computation, vector/matrix operations, and various numerical methods including gaussian elimination, matrix factorizations, etc.

Lecture Insight [Github]

Lecture Insight is a system for converting lecture videos into search-able entities so students can find the content they need faster and in an organized manner.

MetaJ/iJava [Github]

MetaJ/iJava is a project that emerged out of my desire to write java code on my iPhone. However, I couldn't find a free app that would let me develop with a Java IDE and support the functionality I desired. I decided to write a hybrid mobile app using PhoneGap/Cordova that would POST source code to a Java server. The server compiles/invokes the code and sends standard out back to the mobile app. It's open source, and free for anyone to deploy on their own servers and start developing with Java on their iOS devices.
